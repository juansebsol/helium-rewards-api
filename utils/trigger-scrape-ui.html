API usage

List tracked devices: GET /api/manage-devices

List tracked devices (+stats): GET /api/manage-devices?include_stats=true

Add tracked device: POST /api/manage-devices

Remove tracked device: DELETE /api/manage-devices

Trigger Scraping: POST /api/trigger-scrape

Query Rewards: GET /api/helium-rewards?device_key=...&days=30

Date Range Query: GET /api/helium-rewards?device_key=...&start=2025-01-01&end=2025-01-31

Reward Type Filter: GET /api/helium-rewards?device_key=...&reward_type=mobile_verified

Wallet device lookup (Helium): GET https://entities.nft.helium.io/v2/wallet/<walletPublicKey>
